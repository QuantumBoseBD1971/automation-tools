A desktop application for visually editing address blocks in PDF documents.

---

## 🧾 Overview

The PDF Address Editor is designed to simplify the process of correcting or updating address information in invoices and similar documents.

Instead of manually editing PDFs, users can:

- open a PDF file
- draw a rectangle over the address area
- preview the replacement text
- save a corrected version of the document

---

## 🚀 Key Features

- 📂 Open and preview PDF documents
- 🖱 Draw a selection box over the address area
- 👀 Live preview of replacement text
- 💾 Save a new corrected PDF
- 🖥 Desktop application (Windows)
