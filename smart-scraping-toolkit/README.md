## 🧾 Overview

This module represents the web scraping component of the automation toolkit.

It focuses on building reusable scraping utilities capable of handling both simple and complex websites.

---

## 🚀 Core Capabilities

- 🌐 Static scraping (HTML parsing)
- ⚡ Dynamic scraping (JavaScript-rendered pages)
- 🔄 Strategy-based scraping logic
- 📊 Structured data extraction (JSON / CSV)
